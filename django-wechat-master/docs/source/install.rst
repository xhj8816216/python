.. install


============
如何安装
============


通过pip进行安装
-----------------------

- pip install django-weixin

- 在你的 *settings.py* 文件的 *INSTALLED_APPS* 中添加 *django_weixin*


直接使用Github上的项目进行开发
------------------------

- 复制 *settings_eg.py* 为 *settings.py*

- 在 *settings.py* 文件中将 ``APP_ID`` , ``APP_SECRET`` , ``AES_KEY`` , ``WX_TOKEN`` 修改为你自己公众号的信息
